## Overview
This project detects smiles in real-time using a webcam.  
It uses **Python** and **OpenCV** to identify faces and detect when someone is smiling.

---

## Tools & Libraries
- Python 3.x
- OpenCV (`cv2`)
- NumPy

---

## Features
- Real-time face detection
- Smile detection using Haar Cascade classifier
- Displays a message when a smile is detected
- Works with live webcam feed
